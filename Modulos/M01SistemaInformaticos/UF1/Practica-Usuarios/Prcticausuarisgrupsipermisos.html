Usuaris i grups

1. Al servidor amb Linux, crea els grups: empresa, direccio, imesd, produccio i vendes.

        sudo addgroup empresa

        sudo addgroup direccio

        sudo addgroup imesd

        sudo addgroup produccio

        sudo addgroup vendes

2. Crea al servidor Linux els usuaris que es descriuen a la taula anterior amb la comanda adduser i, un cop creats, fes que el seu grup principal sigui el què correspon al seu departament i que tinguin com a grups secundaris els altres departaments als quals pertany més el grup empresa. El grup principal de Gerard ha de ser imesd.

Exemple: Quan creem l’usuari pilar, formarà part del grup principal pilar. A continuació, fem que el seu grup principal sigui vendes i el seu grup secundari sigui empresa.

Creación de los usuarios

sudo adduser peret

sudo adduser gerard

sudo adduser arianda

sudo adduser biel

sudo adduser pilar

sudo adduser cristian

Asignación de los usuarios a los grupos

sudo usermod peret -g direccio -G empresa

sudo usermod gerard -g imesd -G empresa,direccion

sudo usermod ariadna -g produccio -G empresa

sudo usermod biel -g produccio -G empresa

sudo usermod pilar -g vendes -G empresa

sudo usermod cristian -g vendes -G empresa

3. S’acaba de crear el departament de marketing i han contractat a manel perquè en formi part. Crea primer el grup. A continuació crea l’usuari amb la comanda useradd i fes que tingui:

  • Com a grup principal marketing i com a grups secundaris empresa i vendes
  • La seva carpeta personal ha d’estar a /usuaris/manel.
  • El seu intèrpret de comandes per defecte a de ser Bash.
  • Assigna-li una contrasenya i fes que l’hagi de canviar un cop faci el primer login al servidor (cerca la comanda chage).

        Creación del grupo

sudo addgroup marketing

        Creacion del usuario

sudo useradd -g marketing -G empresa,vendes -d /home/manel -m -s  /bin/bash manel

Para obligar a un usuario a que cambie su contraseña una vez inicie sesión usaremos este comando:

sudo chage -d 0 manel

5. Crea a la màquina Linux les carpetes /documents/empresa,  /documents/direccio i /documents/tothom.

sudo mkdir -p /documents/empresa

sudo mkdir  /documents/direccio

sudo mkdir /documents/tothom

Fent servir tant la notació simbòlica com octal, fes que:

  • a la carpeta /documents/direccio només puguin escriure els usuaris peret i gerard i la resta no puguin fer res (ni tan sols accedir).

Cambiamos el grupo propietario de la carpeta  al grupo direccio con el comando:

sudo chgrp direccio direccio

 Luego usamos el siguiente comando para darle el unico permiso que no tiene el grupo: sudo chmod g+w direccio

Y ahora quitamos todos los permisos que aún conservan el resto de usuarios con este comando:

 sudo chmod o-rx direccio

7. Esbrina com eliminar un usuari d’un grup amb línia de comandes. Fes la comprovació eliminant l’usuari biel del grup producció.

Un usuario no puede estar sin grupo principal. Y por esto para eliminar a un usuario de un grupo lo tienes que cambiar de grupo.
