La pràctica consisteix en dues parts: d’una banda instal·larem un servidor Debian amb LDAP i, a la segona part, instal·larem un servidor Active Directory amb Windows Server.

Fes el què et diu la pràctica mirant de fer captures de pantalla que demostrin que has completat cadascuna de les pases necessàries.

Ens han encarregat l’administració de la redacció d’un diari anomenat Diari Nacional (domini: diarinacional.local) que té seus a Barcelona i Girona.

Hem de tenir una estructura on podem configurar tant usuaris com grups a qualsevol de de les seus.

Ahora configuraremos el lam editando el fichero lam.conf que está en la ruta

En este fichero deberemos modificar las siguiente líneas:

types: suffix_group: ou=groups,dc=diarinacional,dc=local

Con esto configurado ya podremos acceder a la página de configuración  del LDAP mediante el navegador usado la siguiente URL:  http://localhost/lam

  • LDAP on tindrem:
  • Les dues seus (cadascuna una OU)
  • Les OUs corresponents a usuaris i grups de cada seu.
  • Els grups editors i redactors.
  • El nostre nom d’usuari (nom.cognom) que pertanyi a la seu de Barcelona i sigui del grup redactors.

Per exemple,

un usuari de Barcelona tindria un DN com el següent:

dn: cn=Ricard Camps,ou=usuaris,ou=barcelona,dc=diarinacional,dc=local

un grup de Girona  tindria un DN com el següent:

dn: cn=editors,ou=grups,ou=girona,dc=diarinacional,dc=local

Para crear las unidades organizativas de cada ciudad deberemos usar dos perfiles de lam, en los cuales configuraremos que los usuarios y grupos creados con ese perfil se creen dentro de la unidad organizativa de la ciudad correspondiente . Para crear estos perfiles en la página  de lam seguimos la siguiente ruta:

LAM Configuration -> Edit Server Profiles -> Manage Server Profiles.
